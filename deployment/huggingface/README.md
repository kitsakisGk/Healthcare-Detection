# AI-Powered Pneumonia Detection

Multi-class chest X-ray classification system powered by deep learning.

## Features

- **4-Class Detection**: NORMAL, BACTERIAL, VIRAL, COVID-19
- **99.14% F1 Score**: State-of-the-art ensemble model
- **Grad-CAM Visualization**: Explainable AI heatmaps
- **Production-Ready**: Clinical deployment ready

## Model Performance

| Model | F1 Score | Parameters |
|-------|----------|------------|
| Ensemble | 99.14% | 876M |
| EfficientNet-B3 | 98.18% | 132M |
| ResNet152 | 97.53% | 661M |

## Technology Stack

- PyTorch 2.0+
- Streamlit
- Grad-CAM
- EfficientNet-B3, ResNet152, DenseNet201
